# Examen #1 - Reportes y Estadísticas

Este proyecto contiene consultas SQL para analizar el rendimiento de vendedores y ventas de propiedades.

## Tablas utilizadas
- vendedores
- propiedades
- ventas
- clientes

## Consultas

1. Cantidad de propiedades vendidas por vendedor
2. Propiedades con precios entre 150M y 400M
3. Clientes con nombre "Carlos"
4. Vendedores con y sin ventas
5. Vista resumen de ventas
